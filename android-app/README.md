# Achtlet Android prototype

This directory contains the original Expo-based Android prototype for Achtlet.

It is not the supported release target. The maintained mobile experience is the
installable PWA in the root app, which uses the same backend and is designed to
work well on Android home screens.

## Status

- Archived for reference
- Not part of the release-ready surface
- May contain outdated assumptions about authentication and backend URLs

## Recommendation

Use the root application instead:

1. Start Achtlet from the repository root.
2. Open it in Chrome on Android.
3. Install the PWA from the browser or the in-app prompt.
