## LinkedIn Post

I just published Achtlet v0.1.0.

Achtlet is an unofficial self-hosted mobile PWA for n8n. It is built for the
small everyday admin jobs I kept wanting on my phone: checking workflow status,
toggling workflows, inspecting recent executions, and creating backups without
opening the full n8n editor.

Release focus:

- Android-friendly installable PWA
- Docker setup for self-hosting
- password-gated UI with server-side sessions
- backup and execution views
- CI, CodeQL, Dependabot, secret scanning, and Docker checks
- clean public release history

The code is MIT licensed and published as-is. I did what I could to test and
harden it, but there is no warranty. Please review it, adapt it to your own
setup, and rerun the checks before trusting it with an important n8n instance.

Repo: https://github.com/grzgrzgrzgrzgrz/achtlet
Release: https://github.com/grzgrzgrzgrzgrz/achtlet/releases/tag/v0.1.0

Achtlet is not affiliated with, endorsed by, or sponsored by n8n GmbH.
